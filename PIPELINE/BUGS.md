# PIPELINE — Bugs & Known Issues

## Известные ограничения

### Концепция не полностью реализована
Kanban Dispatcher и worker profiles — в процессе реализации. Текущая система работает в ручном режиме через Hermes Agent.

### Зависимость от Router 3270
Если Router недоступен, Orchestrator не может работать. Нет автоматического fallback на другой провайдер для оркестрации.

### Codex — дорогой ресурс
Необходимо строго следить за расходом токенов. Использовать только по правилам эскалации.
